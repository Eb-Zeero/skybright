How to use SkyBright



  1. Intoduction


    2. Skybright is data simplifiying software that will display sky brightness of Sutherland. With Skybright you can view the sky brightness of a filter choice, compare it with other filters of any day the observation was made. Hover mouse over a point to see date and time, magnitude and error, and telescope and filter it belong to. By hovering over a median line you will see median of available points of a selected filter, and its minimum value and maximum value



  2. Start page


    3. When SkyBright web is opened. a page just like in Figure 1 below will display.


    1. Chart tabs

      • This controls what tab to be viewed either Date tab or Date range tab
        Date tab:
        This tab displays data of a single date by default. And on load it will load data of the observation of a night before. Also display data of seven days back from the selected date by just clicking a week button. and you can zoom in days to see fewer days of that week.

        Date range:
        This will display by default data of 30 day from the night before. Only selecting a median data point of each observed date.

    2. Date selector

      • Year
        This select a year you wish to view data of
      • Month
        This select a month of a year you wish to view data of
      • Date
        This select a day of a month you wish to view data of

    3. Status chart

      • When data is loading status chart will show progress massage if data is successfully loaded it notify the user and remove the massage, else it will display massage of why chart are empty

    4. Bokeh charts

      • This is where the charts are displayed for the components selected. Charts are divided into five, which are five positions recorded in the server. Each positin name is used as a tittle of a chart. Charts support mouse hover to get details of data you are looking at, and zooming to get a smaller or large scale of data.

    5. Bokeh control tools

      • The pan tool allows the user to pan a Plot by left-dragging a mouse, or on touch devices by dragging a figure or stylus, across the plot region


      • The box zoom tool allows users to define a rectangular region of a Plot to zoom to by dragging he mouse or a figure over the plot region. The end of the drag event indicates the selection region is ready.


      • The wheel zoom tool will zoom the plot in and out, centered on the current mouse location.
        The wheel zoom tool also activates the border regions of a Plot for "single axis" zooming. For instance, zooming in the vertical border or axis will effect a zoom in the vertical direction only, with the horizontal dimension kept fixed.


      • The reset tool is an action. When activated in the toolbar, the tool resets the data bounds of the plot to their values when the plot was initially created.
        The reset tool also resets the plot canvas dimensions to their original size.
        A very useful tool in cases:
        When plots are panned and zoomed, because the axis will remain fixed to the last zoom or pan, even when date is changed. so reset will adjust the view to the default of selected date.


      • The save tool is an action. When activated, the tool opens a download dialog which allows to save an image reproduction of the plot in PNG format. If automatic download is not support by a web browser, the tool falls back to opening the generated image in a new tab or window. User then can manually save it by right clicking on the image and choosing \Save As" (or similar) menu item.


      • Click it to activate or deactivate the hover tool tip hove tool pit is always active on start


      • When activatef it will open a new tab to Boheh homepage.


    6. Cloud coverage slider

      • A slider that will only select data point with cloud coverage less or equal the value selected in percentages. Cloud coverage value is as recorded in the original data on the server.

    7. Filter checkbox

      • Checking a checkbox mean that the data of a filter band represented by the character next to it will be displayed. And unchecked checkbox mean that the data is not displayed on the charts.

    8. Telescope checkbox

      • Data associated with telescope next to the checkbox will be displayed or be removed if the checkbox is checked or unchecked respectively

    9. Exclude moon checkbox

      • If checked data points of all the data of when the moon is up will be excluded from the plot evin from the median of median line calculations.


  3. Date range tab



    1. Chart tabs

      • This is the same as in Start page, Chart tabs
      • All data points of this tab are a median of each observed date between selected range.

    2. Date selector

      • This is the same as in Start page, Date Selector with addition.
      • This consist of two date selectors Start date and End date
        • Start date: This is the start date of the date range you wish to select and in all cases this date must be the oldest in compare to the Start date. Data is selected from moon of this date, morning of this date data will not be selected.
        • End date: This date must be the youngest in compare to the start date and data is selected up until noon of the next day selected, i.e. If end date is 2017/01/01 then the data of 2017/01/02 00:00:00 to 2017/01/02 12:00:00 will be included on the chart

    3. status chart

      • This is the same as in Start page Status chart

    4. Bokeh chart

      • This is where the charts are displayed for the components selected. not like in date tab charts are not divided into five positions. Here the all the positions are displayed on the same chart. where a position can be added or removed from the chart by cheking or unckecking the Positions checkbox. This charts also support mouse hover to get details of data you are looking at, and zooming to get a smaller or large scale of data.

    5. Bokeh control tools

      • This is the same as in Start page Bokeh control tools

    6. Cloud coverage slider

      • This slider will remove all data point that have cloud coverage greater than slider value in percentage. i.e all the data point having cloud coverage will not be included on calculating the median of selected days

    7. Telescope Checkbox

      • This is the same as in Start page Telescope checkbox.

    8. Position checkbox

      • Activating checkbox add to the chart data points of position labeled next to it.

    9. Filter checkbox

      • This is the same as in Start page Filter checkbox.

    10. Exclude moon

      • Activating this will exclude the data point of then the moon is up from calculating the median point of the date.


  4. Date tab chart



    • Position name This is the tittle of the chart representing the position there are four more position on the top of each chart. Namely South, East, North and West.
    • Moon up The section of when the moon is up is shaded with a transparent green color.
    • Moon down The section on the chart, of where the moon is down has no shadings.
    • Median Line The middle line on the chart represent the median point of data points. The color of the line mean the line belong to the data points with the same color.
    • Data Point shapes and color varies thought telescopes.


  5. Date Range tab chart


    6. Data points on this chart are a median of each observed night. That is the median of data taken from the noon of the current date to the noon of the next day. And data points median will be calculated as of what is selected. Thus point that will be considered will be the once that have the values lower or equal the slider value. And if exclude moon checkbox is active then the points of when the moon is up will not be included on the median calculations.


    • Data Points and Legend Data points shapes and color varies thought filters, shapes are as shown in legend. each data point of the same telescope, position and filter band has the same color.
    • Median Line The mean line will represent data point with the same color as it is.
